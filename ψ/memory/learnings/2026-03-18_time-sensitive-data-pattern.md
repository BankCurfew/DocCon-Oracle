# Learning: Time-Sensitive Data Must Have Official Source

**Date**: 2026-03-18
**Source**: Buyer's Guide review — Health Happy "ปิดรับ 31 มี.ค." unverified

## Pattern

ข้อมูล time-sensitive (ปิดรับสมัคร, deadline, วันหมดอายุ, promotion) ต้องมีหลักฐานจาก official source เสมอ ห้ามใส่จากความจำหรือ assumption.

## Why

ถ้า publish ข้อมูล deadline ผิด → ลูกค้าเร่งสมัครโดยไม่จำเป็น หรือพลาดโอกาสจริง → เสียความน่าเชื่อถือ + อาจมีผลทางกฎหมาย (misleading information)

## How to Apply

เมื่อ review content ที่มี:
- วันปิดรับสมัคร / deadline
- ราคาพิเศษ / promotion มีกำหนด
- Policy changes ที่มี effective date
- "ก่อนวันที่ X" / "หลังจากวันที่ Y"

→ **ต้อง flag ทันที** ขอ official source (AIA ประกาศ, email จาก AIA, portal screenshot)
→ ถ้าไม่มีหลักฐาน → CONDITIONAL PASS จนกว่าจะ verify
→ ห้ามให้ FULL PASS content ที่มี unverified deadline

## Evidence

- Writer's Buyer's Guide line 50: "Health Happy จะปิดรับสมัคร 31 มี.ค. 2026"
- AIA verify: ไม่พบหลักฐาน — status=ACTIVE ใน KB, ไม่มี email/ประกาศ
- แบงค์ confirm: "ข้อมูล time-sensitive ต้องมีหลักฐานจาก AIA official เสมอ ห้ามใส่จากความจำ"
