## 1. RECRUITMENT QUESTIONS — ภาษา Professional ไม่ Bias

### 1.1 Language Quality
- [ ] ทุกข้อใช้ภาษาสุภาพ professional
- [ ] ไม่มี typo / grammar errors (TH + EN)
- [ ] จบทุกข้อความด้วย **"ครับ"** ตาม V. Pisit voice
- [ ] ไม่มี hype language ("revolutionary", "game-changing", "ดีที่สุด")
- [ ] ใช้ "แนะนำ" ไม่ใช่ "ต้อง" | "วางแผน" ไม่ใช่ "ซื้อ" | "คุ้มค่า" ไม่ใช่ "ราคาถูก"

### 1.2 Bias & Discrimination Check
- [ ] ไม่ถามอายุ/เพศ/ศาสนา/ชาติพันธุ์ โดยตรง
- [ ] ไม่มี leading questions ที่บังคับคำตอบ
- [ ] ไม่ตัดสินตามอาชีพ — ถามพฤติกรรมไม่ใช่ตำแหน่ง
- [ ] ทุก path ให้กำลังใจ — ไม่มีใคร "ตก" หรือ "ไม่ผ่าน"
- [ ] "ยังไม่ใช่ตอนนี้" ก็มี path ที่ดี + เก็บ contact
- [ ] Persona-based scenes ไม่ดูถูกอาชีพใด (ออฟฟิศ/ร้านค้า/มหาลัย/ห้องหมอ)

### 1.3 Question Style (ตาม Writer concept)
- [ ] ใช้สถานการณ์ถามอ้อม ไม่ถามตรงๆ ("เหมือนดูดวง")
- [ ] Empathize ก่อน advise ตาม V. Pisit philosophy
- [ ] ไม่กดดัน ไม่ใช้ fear tactics / urgency push
- [ ] ทุกตัวเลือกเท่าเทียมกัน ไม่มีคำตอบ "ดี" หรือ "แย่"

## 2. CAREER PATH DESCRIPTIONS — ข้อมูลถูกต้อง ไม่โอ้อวด

### 2.1 18 Career Paths — Accuracy
- [ ] ชื่อ path ตรงกับข้อมูล AIA official (FA Prime / FA Standard / FA 4.0)
- [ ] **FA Prime** vs **FA Standard** — คุณสมบัติถูกต้อง (IC License, ป.ตรี, ประสบการณ์)
- [ ] **CAB** ตัวเลขตรงกับ AIA official PDF (5M/5ปี, 11M+/10ปี)
- [ ] Career ladder ถูกต้อง: Agent → UM → DM → DD → EDD
- [ ] MDRT threshold ถูกต้อง (~FYC ฿982K/ปี)
- [ ] รายได้รับรอง ตรงกับ HR data (30K-150K/เดือน ตาม path)
- [ ] FA 3.5 vs FA 4.0 — แยกชัด (3.5 มีเงินรับรอง 25K, 4.0 เป็น performance-based)
- [ ] IC License requirement ระบุชัด (ต้องสอบผ่าน ก.ล.ต.)

### 2.2 No Overclaim
- [ ] ไม่ใช้คำว่า **"การันตี" / "รับรอง" / "แน่นอน"** กับรายได้
- [ ] รายได้เขียนเป็น **"โอกาสรายได้"** หรือ **"ขึ้นกับผลงาน"**
- [ ] CAB มีเงื่อนไข retention/production — ต้องระบุ "ไม่ได้อัตโนมัติ"
- [ ] ไม่เปรียบเทียบกับคู่แข่ง AIA โดยตรง (กฎ คปภ.)
- [ ] ตัวเลขไม่ปัดเกิน 5% จาก source
- [ ] ระบุ source + date ทุกตัวเลข: "ข้อมูล ณ [date]"

## 3. CTA + DISCLAIMER — ต้องมีครบ

### 3.1 Disclaimers (MANDATORY ทุก result page)
- [ ] **รายได้ disclaimer**: "รายได้ขึ้นกับผลงานจริง ไม่ใช่การรับรองรายได้"
- [ ] **CAB disclaimer**: "CAB มีเงื่อนไข production + retention ที่ต้องผ่านทุกปี"
- [ ] **IC License disclaimer**: "ต้องสอบผ่าน ก.ล.ต. — ไม่ได้มาฟรี"
- [ ] **General disclaimer** (full version):
  ```
  ข้อมูลในเนื้อหานี้เป็นข้อมูลทั่วไป ณ วันที่เผยแพร่ ไม่ถือเป็นคำเสนอขายหรือสัญญาประกันภัย
  รายละเอียดเงื่อนไขและข้อยกเว้นเป็นไปตามกรมธรรม์ กรุณาศึกษารายละเอียดก่อนตัดสินใจ
  ```
- [ ] **ไม่ใช่สัญญาประกันภัย** — ต้องระบุชัด
- [ ] **เบี้ยขึ้นกับอายุ เพศ สุขภาพ และแผนที่เลือก** (ถ้ามีข้อมูลเบี้ย)

### 3.2 CTA (Call to Action)
- [ ] CTA ใช้ **"ปรึกษาฟรี"** ไม่ใช่ "สมัคร" หรือ "ซื้อ"

- [ ] Signature block ครบ: `V. Pisit | Strategic Wealth Curator`
- [ ] Brand mantra **"จริงใจ ลงมือ อยู่ด้วยกัน"** อยู่ใน Epilogue/Result
- [ ] iAgencyAIA.com link
